# docker-xmrMiner

A Docker image for running [xmrMiner](https://github.com/xmrMiner/xmrMiner) based on Ubuntu 16.04 x86_64. Uses the CUDA 8.0 toolkit.

- [xmrMiner GitHub](https://github.com/xmrMiner/xmrMiner)
